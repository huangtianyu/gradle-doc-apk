# Gradle用户指南
Gradle UserGuide apk，支持 android 2.1以上。

Gradle用户指南，可以在地铁或路上随时查看Gradle双语文档。

目前中文翻译还在继续，共享此应用只为方便其他开发者。

应用下载地址：http://www.wandoujia.com/apps/com.githang.gradledoc
